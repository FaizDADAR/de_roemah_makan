# Branding & Design System

Dokumen ini menjelaskan identitas visual dan sistem desain yang digunakan dalam aplikasi De Roemah Makan untuk menjaga konsistensi pengalaman pengguna.

## Konsep Visual
**"Modern Heritage"**: Menggabungkan estetika tradisional masakan nusantara dengan elemen UI modern yang bersih, transparan (*glassmorphism*), dan dinamis.

## Skema Warna (Color Palette)

| Name | Hex | Usage |
| :--- | :--- | :--- |
| **Primary Brown** | `#8B5E3C` | Navbar, Buttons, Primary Accents |
| **Dark Wood** | `#6B3F1F` | Gradients, Dark Backgrounds |
| **Sand Beige** | `#F0E8DF` | Borders, Secondary Backgrounds |
| **Terracotta** | `#A0522D` | Hover states, Highlights |
| **Success Green** | `#16A34A` | Status Success, Confirmations |
| **Error Red** | `#C0392B` | Alerts, Notifications |

## Tipografi
Aplikasi ini menggunakan font sistem yang dioptimalkan untuk keterbacaan:
- **Headings**: `Sans-serif` (Bold) dengan tracking rapat.
- **Body**: `Sans-serif` (Medium/Light) dengan line-height yang nyaman (1.6).

## Komponen UI Khas
- **Glassmorphism**: Digunakan pada navbar dan food cards dengan `backdrop-blur` dan transparansi tinggi.
- **Smooth Gradients**: Transisi warna halus dari `6B3F1F` ke `8B5E3C` untuk memberikan kesan premium.
- **Interactive Hover**: Animasi transisi 300ms pada setiap elemen klik.

## Aset Gambar
Seluruh gambar menu disimpan di:
`public/images/menu/`
Format yang disarankan: `.jpg` atau `.webp` dengan resolusi minimal `800x600px` dan rasio `4:3`.
